# REL_vecteurs_python

#Python #vecteurs #physique #Tracker #seconde #mouvement

Il s’agit d’un programme en Python permettant de visualiser des vecteurs (déplacement, vitesse) à partir de positions relevées à différents instants.
Ces positions peuvent être obtenues grâce à un logiciel de pointage vidéo comme Tracker.
L’objectif est de rendre plus concrètes des notions souvent abstraites en physique, en permettant aux élèves de relier une situation réelle (vidéo) à une représentation vectorielle.

Comment utiliser le code ?
1. Récupérer les données
Filmer un objet en mouvement (ex : balle, voiture, etc.)
Importer la vidéo dans un logiciel de pointage video (Par exemple Tracker)
Relever les positions de l’objet à intervalles réguliers
Exporter les coordonnées x et y

Ouvrir le fichier .py
